Inheritance Sandbox

"All objects in JavaScript inherit from at least one other object."

https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Working_with_Objects

If we create an empty object and look in the console at the prototype object (__proto__) of the object we can see everything it inherits. All objects have these properties including arrays and functions. Functions also inherit from function.prototype and arrays also inherit from array.prototype
