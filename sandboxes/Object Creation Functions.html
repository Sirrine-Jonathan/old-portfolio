Object Creation Functions Sandbox

"Objects can be initialized using new Object(), Object.create(), or using the literal notation (initializer notation)."

https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Object_initializer

Each object is created and passed a variable holding a value to populate them with a key value pair. Then it is stringified and added to the document. You can also see each object in the console. While each is given the variable to make a key value pair, only the object Constructors (new Object() and Constructor()) show the variable used as a key value pair in both the string in the document and in the console.

Object.create() showed no signs of the variable in the object in console unless explicitly accessed, and the the key value pair in the Literal Notation only doesn't show up in the stringified object.
